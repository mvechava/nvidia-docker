1. Login to AWS console
2. Launch instance
	2.1. Click Launch Instance
	2.2. Choose SUSE Linux Enterprise Server 12 SP1 (HVM), SSD Volume type
	2.3. Select either g2.2xlarge or g2.8xlarge instance type depending on HW needs
	2.4. Click Configure Instance Details
		Number of instances - <desired #>
		Network - vpc-bcc003d9 (172.30.0.0/16)
		Subnet - subnet-ea844e9d(172.30.0.0/24) | ap-southeast-2a
		Auto-assign Public IP - Use subnet setting (Enable)
		Placement group - No placement group
		IAM role - None
		Shutdown behavior - Stop
		Enable termination protection - unchecked
		Monitoring - unchecked
		EBS-optimized instance - unchecked
		Tenancy - Shared
		Network interfaces - leave defaults
	2.5. Click Add Storage
		Leave Defaults
	2.6. Click Tag Instance
		Leave Defaults
	2.7. Click Configure Security Group
		Choose default VPC security group
	2.8. Click Review and Launch
		Review preferences
	2.9. Click Launch
		Choose existing key pair
	2.10. Click on launching instance
3. Login and configure instance
	3.1. ssh -i .ssh/<key-name.pem> ec2-user@<public-ip>
	3.2. sudo zypper rr nVidia-Driver-SLE12
	3.3. sudo zypper patch
	3.4. sudo zypper patch (run twice in a row)
	3.5. sudo shutdown -r now
	3.6. ssh -i .ssh/<key-name.pem> ec2-user@<public-ip>
	3.7. wget http://us.download.nvidia.com/XFree86/Linux-x86_64/367.57/NVIDIA-Linux-x86_64-367.57.run (or whichever is the latest driver)
	3.8. sudo zypper in kernel-source gcc make
	3.9. chmod +x NVIDIA-Linux-x86_64-367.57.run
	3.10. sudo ./NVIDIA-Linux-x86_64-367.57.run
		3.10.1. Accept
		3.10.2. Hit OK
		3.10.3. Do not install 32-bit compatibility libaries
		3.10.4. Hit No
		3.10.5. Hit OK
	3.11. sudo shutdown -r now
	3.12. ssh -i .ssh/<key-name.pem> ec2-user@<public-ip>
	3.13. sudo zypper in sle2docker sles12sp1-docker-image git
	3.14. sudo systemctl enable docker.service
	3.15. sudo systemctl start docker.service
	3.16. sudo usermod -aG docker ec2-user
	3.17. sudo sle2docker activate sles12sp1-docker.x86_64-1.0.5-Build10.18
	3.18. sudo su -
	3.19. cd /var/lib/docker/image
	3.20. git clone https://github.com/mvechava/nvidia-docker.git
	3.21. cd nvidia-docker
	3.22. echo sles12sp1/cuda/7.0/devel/ sles12sp1/cuda/7.5/devel/ sles12sp1/cuda/8.0/devel/ samples/sles/bandwidthTest/ samples/sles/deviceQuery/ samples/sles/matrixMulCUBLAS/ samples/sles/nbody/ samples/sles/vectorAdd/ | xargs -n 1 cp -rf /etc/zypp /etc/hosts
	3.23. make cuda OS=sles12sp1 (this will take awhile)
	3.24. make tools
	3.25. make install tools
	3.26. make samples
	3.27. nohup nvidia-docker-plugin > /tmp/nvidia-docker.log 2>&1 &
	3.28. nvidia-docker run --rm sample:nvidia-smi
